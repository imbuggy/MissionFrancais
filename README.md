# Mission Français CE1 🚀🇫🇷

Une application mobile-first complète pour apprendre la conjugaison, les nombres et la grammaire en français pour les élèves de CE1.

## ✨ Fonctionnalités

- **Conjugaison** : Présent, Passé Composé et Futur Proche (Être, Avoir, Aller, Faire, Dire, Venir et 1er groupe).
- **Nombres** : Apprendre à écrire les nombres en toutes lettres (0-9, 10-99, 100-999).
- **Grammaire** : Genre (Masculin/Féminin) et Nombre (Singulier/Pluriel).
- **Gamification** : Système de combo, effets sonores et animations de fusée.
- **Suivi de Progression** : Système de maîtrise (3 réussites pour valider un item) et page de progression dédiée.
- **Mobile-First** : Conçu pour fonctionner parfaitement sur téléphone (PWA installable).

## 🚀 Installation Locale

1. Clonez le dépôt :
   ```bash
   git clone <votre-repo-url>
   cd mission-francais-ce1
   ```

2. Installez les dépendances :
   ```bash
   npm install
   ```

3. Lancez le serveur de développement :
   ```bash
   npm run dev
   ```

## 📦 Déploiement sur GitHub Pages

1. Construisez le projet :
   ```bash
   npm run build
   ```

2. Déployez le contenu du dossier `dist` sur la branche `gh-pages`.

## 📱 Installation sur Mobile (PWA)

L'application est une **Progressive Web App**. Pour l'installer :
- **Sur iOS (Safari)** : Appuyez sur le bouton "Partager" et choisissez "Sur l'écran d'accueil".
- **Sur Android (Chrome)** : Appuyez sur les trois points et choisissez "Installer l'application".
